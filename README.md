# BilingualDictionary

**`#Java` `#MVC`**

A simple dictionary with local database.

## App features
- 4000 words
- Delete and reset database
- Add new word
- Editable words
- Share Words
- Multi language

## App structure
- Material design
- SQLite database
- Lottie animation
